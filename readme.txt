This is in beta mode, butterflies work.   If you would like to contribute fun scripts to this plugin contact philip@frumph.net

Added Linkfader, makes your links glow green then fade after mouse overing them.
Added 2 new image tricks that have fun with images expanding and 3 balloons

== Installation ==

1. Upload the `theme-tricks` directory to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
1. Go to Appearance -> Theme Tricks to start using the Theme Tricks.
1. Choose whatever effect you want to have displayed on your site.

== Frequently Asked Questions ==

= I'm having another problem. =

File a bug on the [WebComic Planet Forums](http://webcomicplanet.com/forum/theme-tricks/).  Make sure you're running the most recent stable version of Theme Tricks, as there are a lot of critical bug fixes between versions.

If it's a serious problem, such as WordPress becoming non-functional or blank screens/errors after you perform certain operations, you will be asked to provide error logs for your server.  If you don't know how to get access to these, talk with your Webhost.  At that time just remove the plugin from the plugins/ directory.  If it still
continue's it is probably not a Theme Tricks problem.
